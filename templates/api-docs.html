Overview

This API allows you to interact with our service programmatically. You can retrieve, create, update, and delete resources using the endpoints provided.

Authentication

To use the API, you need to authenticate using an API key. Include the API key in the header of your requests as follows:

Authorization: Bearer YOUR_API_KEY

Endpoints

GET /api/v1/resources

Retrieve a list of resources.

Sample Request

curl -X GET "https://api.example.com/v1/resources"

Sample Response

{
    "data": [
        {
            "id": 1,
            "name": "Resource 1",
            "description": "Description of Resource 1"
        },
        {
            "id": 2,
            "name": "Resource 2",
            "description": "Description of Resource 2"
        }
    ]
}

POST /api/v1/resources

Create a new resource.

Sample Request

curl -X POST "https://api.example.com/v1/resources" -d '{
    "name": "New Resource",
    "description": "Description of the new resource"
}'

Sample Response

{
    "id": 3,
    "name": "New Resource",
    "description": "Description of the new resource"
}

PUT /api/v1/resources/{id}

Update an existing resource.

Sample Request

curl -X PUT "https://api.example.com/v1/resources/1" -d '{
    "name": "Updated Resource",
    "description": "Updated description of the resource"
}'

Sample Response

{
    "id": 1,
    "name": "Updated Resource",
    "description": "Updated description of the resource"
}

DELETE /api/v1/resources/{id}

Delete a resource.

Sample Request

curl -X DELETE "https://api.example.com/v1/resources/1"

Sample Response

{
    "message": "Resource deleted successfully"
}

Examples

Example 1: Fetching Resources

fetch('https://api.example.com/v1/resources', {
    method: 'GET'
})
.then(response => response.json())
.then(data => console.log(data));
import requests

response = requests.get('https://api.example.com/v1/resources')
print(response.json())
require 'net/http'
require 'json'

url = URI('https://api.example.com/v1/resources')
response = Net::HTTP.get(url)
puts JSON.parse(response)

Example 2: Creating a Resource

fetch('https://api.example.com/v1/resources', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        name: 'New Resource',
        description: 'Description of the new resource'
    })
})
.then(response => response.json())
.then(data => console.log(data));
import requests

data = {
    'name': 'New Resource',
    'description': 'Description of the new resource'
}
response = requests.post('https://api.example.com/v1/resources', json=data)
print(response.json())
require 'net/http'
require 'json'

url = URI('https://api.example.com/v1/resources')
http = Net::HTTP.new(url.host, url.port)
request = Net::HTTP::Post.new(url)
request['Content-Type'] = 'application/json'
request.body = {
    name: 'New Resource',
    description: 'Description of the new resource'
}.to_json
response = http.request(request)
puts JSON.parse(response.body)
